# YOLO-Attendance-System
Automated Attendance System using YOLOv5 and Deep Learning


## 📖 Project Overview

This project automates Attendance marking from group photographs using Deep Learning and Computer Vision techniques.

The system detects and identifies students from a classroom Image and Automatically generates Attendance records.

## 🎯 Objectives

- Reduce manual attendance effort
- Improve attendance accuracy
- Enable contactless attendance management
- Apply Deep Learning in real-world applications

## 🛠 Technologies Used

- Python
- YOLOv5
- OpenCV
- NumPy
- Pandas
- Deep Learning

## ⚙️ System Architecture

Group Photo
↓
YOLOv5 Detection
↓
Student Identification
↓
Attendance Generation
↓
Attendance Report

## 🚀 Features

- Automatic Student Detection
- Attendance Generation
- Group Photo Processing
- Fast Inference
- Deep Learning-Based Recognition
